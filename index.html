La seguridad es la prioridad

Portal de Gestión Interna

Bienvenido al sistema de Hospital Serena. Este portal es exclusivamente para el uso de el personal, con el objetivo de administrar la atención de nuestros pacientes.

La pagina contiene los siguientes sectores:

  • Pacientes: Control de hospitalización, camas asignadas, diagnósticos, historial de visitas y estado clínico.
  • Personal: Directorio médico.
  • Citas: Agenda de consultas programadas y asignación de consultorios.
  • Urgencias: Monitoreo de emergencias clasificado bajo prioridad.
  • Farmacia: Inventario de existencias y control de medicamentos.

Puedes usar la barra de navegación para acceder a los diferentes sectores.
